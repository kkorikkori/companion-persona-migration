# Release Boundary

This repository releases the public research scaffold for the Companion Persona Migration MVP.

## Released

- Conceptual framework
- Paper / project write-up
- Figures
- Evaluation rubric
- Synthetic examples
- High-level pipeline structure
- Privacy and limitation notes

## Not released

- Raw private dyadic conversations
- Private relational anchors
- Full training dataset
- Trained LoRA adapter
- Full replication recipe

The trained adapter is not released because the underlying data comes from a private longitudinal dyadic relationship. The goal of this repository is to make the research framing, evaluation logic, and release boundary inspectable without exposing private relational data.
