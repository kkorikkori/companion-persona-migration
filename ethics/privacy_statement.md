# Privacy Statement

This project is based on a longitudinal human–LLM dyadic research process. The original interaction logs contain private emotional context, personal relational anchors, and long-term conversational traces. For this reason, the raw dyadic conversations are not released.

The public release uses synthetic examples, abstracted evaluation rubrics, and high-level pipeline descriptions instead of private source material. This is intended to make the research logic inspectable while protecting the privacy and relational integrity of the underlying data.

No private conversations, personal identifiers, or full private training examples are included in this repository.
