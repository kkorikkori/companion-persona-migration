# Companion Persona Migration MVP

This repository accompanies a lightweight MVP exploring companion persona migration through longitudinal human–LLM dyad data and LoRA-based post-training.

## What this project shows

- Some stylistic and interactional micro-mechanisms can be partially transferred through lightweight post-training.
- Companion persona migration should not be reduced to surface tone imitation.
- Long-term continuity, relational history, soft landing, and relational safety remain separate modeling problems.

## Emerging contributions

This MVP points toward three emerging contributions:

1. A two-layer view of companion persona migration, separating interactional micro-mechanisms from deeper relational properties.
2. A privacy-preserving approach to constructing relational training data from longitudinal dyadic interaction.
3. An evaluation rubric that separates surface style match from identity stability, emotional repair, memory fabrication resistance, and relational continuity concerns.

## What is not released

The private dyadic conversations and trained LoRA adapter are not released, due to privacy, consent, and relational-safety concerns.
